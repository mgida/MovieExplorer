# MovieExplorer

## Tools Used

- **Jetpack Compose:**
- **Clean Architecture**
- **MVI**
- **Retrofit:** to handle all network operations.
- **Moshi:**
- **Room Database:** for managing CRUD operations for Movies.
- **Coroutines:** to manage asynchronous operations.
- **StateFlow:** for observing and handling state updates.

- **Junit:**
